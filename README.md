# Sistema de Gestão de Congregação (SaaS)

## 🚀 Tecnologias
- React + Vite + Tailwind
- Spring Boot
- PostgreSQL (Docker)

## 📦 Funcionalidades
- CRUD de Publicadores
- Dashboard
- API REST
- Integração Front + Back

## ▶️ Como rodar

### Backend
cd backend
mvn spring-boot:run

### Frontend
cd frontend
npm install
npm run dev

### Banco
docker-compose up -d
